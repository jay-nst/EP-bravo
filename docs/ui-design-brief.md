# EarthPaper UI Design Brief

> Figma 프로토타입 기준: https://www.figma.com/proto/l4wvGeL6vN5TZZiSP4DvaU/어스페이퍼_Jay?node-id=17-142&starting-point-node-id=71:40

## 제품 개요

위성 영상(Observer 1.5m, SpaceEye-T 25cm) 셀프서비스 구매 포털.
로그인 후 포털 대시보드가 메인 화면이며, 오늘의 위성 사진, 트렌딩 위치, 내 주문 현황,
서비스 바로가기를 한눈에 볼 수 있는 B2B SaaS 플랫폼. (Professional Discovery 컨셉)

## 사내 디자인 시스템

Figma: https://www.figma.com/design/7cgOfL2jmPpOiYuLEEeI5o/2026-UI-Design-System?node-id=81-1805
- 위 디자인 시스템의 색상, 타이포, 컴포넌트 스타일을 따라주세요

## 기술 스택

- Next.js + Tailwind CSS
- Mapbox GL JS (위성 지도)
- 다크 테마 기본 (위성 영상 플랫폼 특성)

---

## 핵심 구조 변경: 포털 중심 설계

**현재 문제**: `/`가 마케팅 랜딩 페이지로 되어 있음
**변경**: 로그인 사용자의 메인 화면은 **포털 대시보드** (`/dashboard` 또는 `/`)

```
비로그인 사용자:
  / (랜딩) → /login → /dashboard (포털 대시보드)

로그인 사용자:
  / → /dashboard로 리다이렉트 (포털 대시보드가 홈)
```

---

## 디자인할 화면 목록

### 1. 포털 대시보드 (/dashboard) — 메인 화면 (로그인 후 홈)

**v4.2 Professional Discovery + Discovery Feed** — 게이미피케이션 없이, 다양한 콘텐츠와 서비스 연결 중심의 대시보드.
전체 화면 Mapbox 위성 지도를 배경으로 glass-morphism 패널들이 floating되는 구조.

#### 배경
- 전체 화면 Mapbox 위성 지도 (satellite-streets 스타일)
- 패널 뒤로 자연스럽게 보이는 지구 위성 영상

#### 헤더 오버레이
- **좌측**: "EARTHPAPER" 로고 텍스트
- **중앙**: 서비스 네비게이션 탭
  - 홈(활성) | 지도 | 채팅 | 촬영 요청 | 내 주문
  - 각 탭 클릭 시 해당 서비스 페이지로 이동
- **우측**: 알림 벨(뱃지) + 이메일 + 로그아웃

#### 본문 — 3컬럼 Floating Panels (glass-morphism)

**좌측 패널: 디스커버리 피드 (Discovery Feed)**
- 필터 탭: 전체 | 분석 | 뉴스 | 포스트 | 숏츠 | 트렌딩
- 6가지 콘텐츠 타입이 혼합된 무한 스크롤 피드:

  1. **AI 분석 카드** (badge: "AI 분석")
     - 분석 유형: 변화 탐지, 식생 분석(NDVI), 도시 열섬 등
     - 시각적 표현: 히트맵/그래프 영역
     - 위치명 + 분석 날짜 + "상세 보기" 버튼

  2. **숏츠 카드** (badge: "숏츠")
     - 가로 레이아웃: 썸네일(재생 아이콘 + 재생시간 뱃지) + 제목/설명
     - 타임랩스, 전후 비교 등 짧은 영상
     - 조회수 표시

  3. **트렌딩 위치** (compact list)
     - 순위 번호(1-5) + 위치명 + 좌표 + 촬영일 + 해상도
     - "지도에서 보기" 버튼 → `/map?lat=&lng=&zoom=`

  4. **뉴스/보고서 카드** (badge: "뉴스" / "보고서")
     - 출처 + 게시일 + 제목
     - 카테고리 뱃지 (상업, 기술, 소식, 민간)

  5. **커뮤니티 포스트** (badge: "커뮤니티")
     - 작성자 아바타 + 닉네임 + 작성일
     - 위성 사진 첨부 (위치 태그)
     - 반응: 좋아요 + 댓글 + 공유 버튼

  6. **주간 리포트** (badge: "주간 리포트")
     - 요약 텍스트 + "전체 보기" 링크

**중앙 패널: 오늘의 위성 사진 + 내 주문**
- "오늘의 위성 사진" 카드
  - 고해상도 위성 사진 썸네일
  - 촬영 위치 + 날짜 + 위성명 + 해상도
  - "지도에서 보기" 버튼 → `/map` 이동
- "내 주문" 섹션
  - 최근 주문 카드 목록 (3-4개)
  - 각 카드: 주문번호(앞 8자리) + 면적 + 상태 뱃지
  - 상태별: 대기중(노랑), 처리중(파랑+진행률 바), 완료(초록+다운로드 버튼)
  - 클릭 시 → `/portal` 상세 페이지로 이동
  - "→ 전체 주문 보기" 링크

**우측 패널: 퀵 도구**
- "랜덤 탐험" 버튼 → 임의 좌표로 지도 이동
- AI 채팅 위젯 (미니 인풋)
  - "위성 영상에 대해 물어보세요..." 플레이스홀더
  - 입력 시 `/chat`으로 확장
- 촬영 요청 CTA
  - "새 촬영 요청" 버튼 → `/tasking` 이동
  - 진행 중인 요청 수 뱃지 표시
- 내 계정 요약
  - 보유 영상 수, 총 분석 면적, 가입일 표시
- 관심지역 (Watchlist)
  - 사용자가 설정한 AOI 목록
  - 각 항목: 지역명 + 새 영상 도착 수
  - 클릭 시 → `/map?aoi=` 로 이동

#### 서비스 연결 흐름 (Service Connection)
```
디스커버리 피드 콘텐츠:
  AI 분석   → "상세 보기" → /map (분석 결과 오버레이)
  숏츠     → 클릭 → 인라인 재생 또는 /shorts/[id]
  트렌딩   → "지도에서 보기" → /map (좌표 전달)
  뉴스     → 클릭 → 외부 링크 또는 /insights/[id]
  커뮤니티  → 클릭 → /community/[id] (향후)

중앙/우측 패널:
  오늘의 사진 → "지도에서 보기" → /map
  내 주문    → 클릭 → /portal 상세, 다운로드 직접 가능
  AI 채팅    → 인라인 질문 또는 /chat 확장
  촬영 요청  → /tasking (진행 중 건수 표시)
  관심지역   → 클릭 → /map?aoi=
  랜덤 탐험  → 임의 좌표로 지도 배경 이동
```

---

### 2. 랜딩 페이지 (/) — 비로그인 사용자 전용

- 간단한 제품 소개 + CTA
- 히어로: "위성 영상, 직접 선택하고 바로 다운로드"
- CTA: "시작하기" → /login으로 이동
- 로그인 사용자가 접근 시 /dashboard로 리다이렉트

---

### 3. 로그인 (/login)

- 중앙 정렬 폼: 이메일 + 비밀번호
- "EARTHPAPER" 로고/텍스트 상단
- 로그인 버튼 + 회원가입 링크
- 에러 메시지 영역

### 4. 회원가입 (/signup)

- 중앙 정렬 폼: 이름 + 이메일 + 비밀번호
- 가입 완료 시 "이메일 확인" 안내 화면

---

### 5. 지도 페이지 (/map) — 핵심 기능 화면

- **레이아웃**: 전체 화면 지도 + 우측 사이드바 패널
- **지도 영역**:
  - Mapbox 위성 지도 (satellite-streets 스타일)
  - 좌상단: 폴리곤 그리기/삭제 도구
  - 우상단: 줌 컨트롤
  - AOI 폴리곤이 그려진 상태 표시
- **우측 사이드바** (320px 폭):
  - 위성 선택: Observer / SpaceEye-T 토글/탭
  - 선택된 위성 스펙: 해상도, 초해상도, 가격/km², 최소면적
  - AOI 정보: 면적 (km²), 예상 가격 ($)
  - 유효성 에러 메시지 (면적 부족 등)
  - "구매하기" 버튼 (비활성/활성 상태)
  - AOI 미선택 시: "지도에서 영역을 그려주세요" 안내

---

### 6. 내 주문 포털 (/portal)

- 주문 카드 리스트 (최신순)
- 각 카드:
  - 주문번호 (앞 8자리), 날짜
  - 면적 + 금액
  - 상태 뱃지: 대기중(노랑), 처리중(파랑, 애니메이션), 완료(초록), 실패(빨강), 환불됨(회색)
  - 완료 시: "다운로드" 버튼
  - 실패 시: 에러 메시지 표시
  - 만료 시: "다운로드 기간 만료" 텍스트
- 빈 상태: "주문 내역이 없습니다" + 지도 링크

---

### 7. AI 채팅 (/chat)

- 좌측 사이드바: 채팅 세션 목록 + 새 채팅 버튼
- 메인 영역: 메시지 스트리밍 UI
- 위성 영상 상담 전용 AI

### 8. 촬영 요청 (/tasking)

- 촬영 요청 폼: 날짜, 이메일, 전화번호, 메모
- 요청 목록: 상태 뱃지 포함 (received, reviewing, quoted, accepted, rejected)

---

### 9. 공통 헤더 (네비게이션)

**v4.1 서비스 네비게이션 중심 헤더**:
```
[EARTHPAPER]  [홈] [지도] [채팅] [촬영 요청] [내 주문]    [🔔(3)] [이메일] [로그아웃]
```

- 좌측: "EARTHPAPER" 로고 텍스트 (대문자, tracking-tight)
- 중앙: 서비스 네비게이션 탭 (홈/지도/채팅/촬영 요청/내 주문)
  - 활성 탭: bg-gray-800 text-white 배경
  - 비활성 탭: text-gray-400, hover 시 text-white
- 우측: 알림 벨 아이콘(미읽음 수 뱃지) + 사용자 이메일 + 로그아웃 버튼
- 배경: bg-gray-950/80 backdrop-blur, border-bottom gray-800
- sticky top-0, z-50

---

### 10. 결제 모달/플로우 (Toss Payments)

- 구매 확인 모달: 영상 정보 요약 + 면적 + 최종 가격
- "결제하기" → Toss 결제 위젯 호출
- 처리중 상태: 스피너 + "영상 클리핑 처리 중..."
- 완료: 성공 메시지 + 다운로드 버튼
- 실패: 에러 + 자동환불 안내

---

## 디자인 방향

- **Professional Discovery + Discovery Feed**: 위성 영상 탐색, 다양한 콘텐츠, 서비스 연결 중심
- 다크 테마 기반 + glass-morphism 패널 (위성 지도 배경과 조화)
- 전문적이고 깔끔한 B2B SaaS 느낌 — 게이미피케이션 요소 배제
- 대시보드에서 모든 서비스로의 명확한 진입점 제공 (피드→지도, 주문→포털, 채팅→AI)
- 6가지 콘텐츠 타입(AI 분석, 숏츠, 트렌딩, 뉴스, 커뮤니티, 리포트)으로 클릭 유도
- 지도 페이지에서는 지도가 주인공 — UI는 지도를 방해하지 않게

## 페이지 라우팅 변경

| 경로 | 비로그인 | 로그인 |
|------|---------|--------|
| `/` | 랜딩 페이지 표시 | `/dashboard`로 리다이렉트 |
| `/dashboard` | `/login`으로 리다이렉트 | 포털 대시보드 (메인) |
| `/map` | `/login`으로 리다이렉트 | 지도 서비스 |
| `/portal` | `/login`으로 리다이렉트 | 내 주문 |
| `/chat` | `/login`으로 리다이렉트 | AI 채팅 |
| `/tasking` | `/login`으로 리다이렉트 | 촬영 요청 |

## 상태별 변형 (디자인 필요)

- 버튼: default, hover, disabled, loading
- 인풋: default, focus, error
- 주문 상태 뱃지: pending, processing, completed, failed, refunded
- 촬영 요청 상태 뱃지: received, reviewing, quoted, accepted, rejected
- 사이드바: AOI 미선택 / 선택됨 / 에러
- 알림 벨: 미읽음 있음 / 없음
- 대시보드: 콘텐츠 있음 / 빈 상태 / 로딩 스켈레톤
- Discovery Feed: 필터별 콘텐츠 있음 / 빈 상태 ("해당 콘텐츠가 없습니다")
- Discovery Feed 카드: AI 분석 / 숏츠 / 트렌딩 / 뉴스 / 커뮤니티 / 리포트 (6종)
- 관심지역: 목록 있음 / 빈 상태 ("관심지역을 추가해보세요")
